# react-native-metroipo

Official React-Native wrapper for the Metro Ipo SDK.

## Installation

```sh
npm install react-native-metroipo
```

## Usage

```js
import Metroipo from "react-native-metroipo";

// ...

const result = await Metroipo.multiply(3, 7);
```
